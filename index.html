GitHub sync
Your food logs sync automatically to your private jainomics/gains-data (private) — available on all your devices. Paste your GitHub Personal Access Token once to activate.
Token is stored only in your browser. The app syncs automatically after each food entry. Your token expires in 90 days — you'll see a sync error when it does, at which point generate a new one on GitHub and paste it here.
AI logging secret
The secret key that authenticates requests to your Cloudflare Worker. This is the value stored in your Cloudflare Worker secret named WORKER_SECRET. Paste it here and it will be stored only in your browser.
If AI logging stops working, check this is set. You can change the secret in your Cloudflare Worker settings and update it here to match.
